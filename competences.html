Projets scolaires


Les projets réalisés au sein de l'IUT sont conçus pour être professionnalisant. Ils permettent de développer différentes compétences techniques et humaines.

Régression sur données réelles

Projet statistique de première année réalisé en binôme sur une durée d'une semaine. Nous modélisons statistiquement des données dans l'objectif de mettre en avant un lien entre deux populations.

Réalisation de code en R, Applications de méthodes statistiques, Réponse à une problématique posée.


Intégration de données dans un DW

Projet informatique de deuxième année réalisé en trinôme sur l'ensemble d'un semestre. Nous modélisons puis créons un Système d'Information Décisionnel (SID) à partir de bases de données hétérogènes dans le but de réaliser des analyses et tableaux de bords.

Modélisation de SID, Processus ETL, Visualisation.


Etude d'impact d'une technique médicale

Projet de première année réalisé sur l'ensemble d'un semestre. Description de données et tests statistiques pour répondre à une problématique donnée. Comprends une partie "conduite de projet".

Applications de méthodes statistiques, Organisation d'un projet d'ampleur.
